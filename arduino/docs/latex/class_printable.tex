\subsection{Detailed Description}
The \hyperlink{class_printable}{Printable} class provides a way for new classes to allow themselves to be printed. By deriving from \hyperlink{class_printable}{Printable} and implementing the print\-To method, it will then be possible for users to print out instances of this class by passing them into the usual \hyperlink{class_print_aa4158dd94bc1741f92d99c427261d7c0}{Print\-::print} and \hyperlink{class_print_a4fd286b325d3b1a786cfa35072a8ef52}{Print\-::println} methods.
